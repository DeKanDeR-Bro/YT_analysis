# YouTube Minecraft War Analysis

## О проекте
Аналитическое исследование активности и вовлеченности аудитории YouTube-каналов в нише Minecraft. Проект сравнивает показатели различных групп каналов (FT, LITE, CLASSIC) для выявления аномалий и эффективных стратегий контента.
Файл с получившейся БД залил на гугл диск: https://drive.google.com/file/d/10zMhqvYabLGLoHyjtglYJjFQDQdewS4a/view?usp=drive_link
(из-за лимитов ГХ не закинуть в репозиторий, т.к. файл получился слишком большой)
**Период, за который собирались данные: 01.01.2024 — 01.02.2026

**Данное исследование является первичным и не может полностью доказывать и/или отрицать факт какой-либо накрутки. Для реального подтверждения необходимо проводить privacy-analysis с автоматизированным сбором статистики лично каждым автором канала через собственный API-KEY.**

## Структура репозитория
- `analysis/` - Jupyter ноутбуки и скрипты для статистического анализа и генерации отчетов.
- `collector/` - Скрипты для сбора данных через YouTube Data API v3.
- `requirements.txt` - Список зависимостей.

## Ключевые файлы
- **analysis/report.ipynb** - Интерактивный отчет с визуализацией.
- **analysis/final_report.pdf** - Итоговый PDF отчет с выводами и графиками.
- **analysis/metodic.pdf** - Методическое пособие по метрикам (LVR, CVR, ER).
- **collector/DATA_COLLECTION_METHODOLOGY.md** - Описание методики сбора данных.

## Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/your-username/minecraft-war-analysis.git
   cd minecraft-war-analysis
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Анализ данных:**
   Перейдите в папку `analysis` и запустите Jupyter Notebook:
   ```bash
   cd analysis
   jupyter notebook report.ipynb
   ```

## Используемые технологии
- **Python 3.10+**
- **Pandas** (обработка данных)
- **Seaborn / Matplotlib** (визуализация)
